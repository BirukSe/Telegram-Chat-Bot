🚀 Quick Deployment
Get your Telegram Bot Token from @BotFather

Clone the repo:

bash
git clone https://github.com/yourusername/burak-ai-bot.git
Configure:

bash
cp .env.example .env
nano .env  # Add your BOT_TOKEN
Start the bot:

bash
npm install
npm start
